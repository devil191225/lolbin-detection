# Disclaimer & Legal Notice

## Purpose
This repository implements behavioral detection for living-off-the-land binaries
(LOLBins) — legitimate system tools abused by attackers to evade detection.
It is a **defensive security tool** built to help security teams identify
malicious use of system utilities.

## Authorized Use
- ✅ Deploying detections on systems you own or are authorized to monitor
- ✅ Security research and red team/blue team exercises
- ✅ Academic study of attacker techniques for defensive purposes

- ❌ Using the LOLBin knowledge encoded here to attack systems you do not own
- ❌ Bypassing security controls on unauthorized systems
